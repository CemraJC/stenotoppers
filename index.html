About StenoToppers

A StenoTopper is a 3D printed keycap set that aims to optimize a mechanical keyboard for stenography - when used in conjunction with the Plover software. The set achieves this by raising, aligning and compacting the keys, making it easier to press multiple keys as required. It also moves the steno keys away from the surrounding keys, such as the spacebar, making stenotyping much more comfortable.

StenoToppers and other similar solutions (like the Laser-cut Keytoppers) were designed to counteract the problem of the QWERTY layout. As it stands, QWERTY is designed to make it difficult to press multiple keys at once. Stenographers using Plover on a QWERTY keyboard tend to notice that it's annoying and sometimes painful to be constantly avoiding the spacebar, slanting your fingers to line up with the keyboard's row staggering and bending thumbs and pinkie fingers to overcome the substantial gap between the keys. With StenoToppers, one can align the rows, raise them above the spacebar and move the keys closer in one fell swoop. Reclaim your ergonomic advantage with StenoToppers.

How to get some

Currently, StenoToppers are only available through 3D printing. This is because, although very useful, they are only useful for a relatively small number of people. Demand for units would need to be in the 10,000's to make mass production viable.

If you already have access to a 3D printer, then you should be able to download the models above and print them off for negligible cost (about $2.50 worth of filament, duration of 12 hours). If, like the rest of us, you don't have access to a 3D printer, you will need to utilize a 3D printing service like Shapeways for a minor cost (about $COST_EST). When using Shapeways, you can upload and configure the models yourself or you can use the ones already uploaded for free (well, there is the print cost).
